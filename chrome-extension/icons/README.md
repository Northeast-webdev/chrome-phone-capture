# Extension Icons

## Required Icons

The extension needs three icon sizes:
- `icon16.png` - Toolbar icon (16x16px)
- `icon48.png` - Extension management page (48x48px)
- `icon128.png` - Chrome Web Store (128x128px)

## Temporary Solution

For development/testing, you can use placeholder icons. You can create simple PNG files or use any icon.

## Permanent Icons

For production, design professional icons with:
- Phone/dialer symbol
- Brand colors (blue/green)
- Transparent background
- High quality PNG

## Quick Icon Generation

Use online tools like:
- https://www.favicon-generator.org/
- https://realfavicongenerator.net/
- Or design in Figma/Photoshop

## Icon Guidelines

- **Simple**: Clear at small sizes
- **Recognizable**: Phone/call symbol
- **Consistent**: Same design across sizes
- **Professional**: Matches your brand
